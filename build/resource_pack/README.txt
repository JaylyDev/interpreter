GameTest Interpreter

Documentation: Available on MCPEDL
https://mcpedl.com/gametest-interpreter/

GameTest Interpreter allows Minecraft: Bedrock Edition to execute code written in a programming language.
The add-on currenly supports JavaScript, TypeScript and *BrainFuck*.
